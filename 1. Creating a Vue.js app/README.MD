## 1) Install the vue command line interface

```npm i -g @vue/cli```

## 2) Use the command line interface to create the app

```vue create <name>```

## 3) Cd into the directory created by the CLI

```cd client```

## 4) Run the app for development

```npm run serve```

## 5) Build the app for deployment

```npm run build```
